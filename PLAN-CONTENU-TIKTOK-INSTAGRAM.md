## 1. RECHERCHE ALGORITHME

### TikTok - Comment l'algorithme classe les videos

| Facteur | Poids | Comment l'exploiter |
|---------|-------|---------------------|
| **Taux de completion** (watch time) | #1 | Videos courtes (15-30s) pour garantir 100% completion |
| **Re-watch** (boucle) | Tres eleve | Finir la video avec un element du debut = boucle naturelle |
| **Partages** | Eleve | Contenu "tag ton amie qui fait ca" |
| **Commentaires** | Eleve | Poser une question en fin de video |
| **Likes** | Moyen | Hook fort dans les 2 premieres secondes |
| **Follows apres visionnage** | Moyen | CTA subtil "follow pour la suite" |
| **Hashtags** | Moyen | Mix 3 gros + 3 niches + 1 trending |

### Instagram Reels - Differences cles

- Favorise les comptes qui postent REGULIEREMENT (3-5 Reels/semaine minimum)
- Les Reels avec AUDIO ORIGINAL performent mieux que les reposts TikTok
- Les 3 premieres secondes sont encore PLUS critiques que sur TikTok
- Les Reels de 15-30 secondes ont le meilleur ratio reach/engagement
- Les hashtags comptent encore (10-15 par post)

## 4. SERIES DE CONTENU (10 formats rotatifs)

### Serie 1 : "Elia note tes soins" (3x/semaine)
> Elia analyse les ingredients d'un produit populaire et donne un score /100
- Format : 25-35 secondes
- Hook : "Je note la [MARQUE] [PRODUIT]"
- Structure : Hook (2s) > Liste ingredients (10s) > Verdict (5s) > Score (3s) > CTA (2s)

### Serie 2 : "Ingredients toxiques" (2x/semaine)
> Alerter sur les ingredients a eviter
- Format : 20-30 secondes
- Hook : "Arrete ta creme si tu vois CA"
- Structure : Hook choc (2s) > 3 ingredients a fuir (15s) > Alternative clean (5s) > CTA (3s)

### Serie 3 : "Routine en 60 secondes" (2x/semaine)
> Tutoriels rapides par type de peau
- Format : 45-60 secondes
- Hook : "Routine [type de peau] en 60 secondes"
- Structure : Intro (3s) > Etape 1-4 (10s chaque) > Resultat (5s) > CTA (3s)

### Serie 4 : "VS - Le match" (1x/semaine)
> Comparaison de 2 produits/techniques
- Format : 30-40 secondes
- Hook : "[Produit A] ou [Produit B] ?"
- Structure : Question (3s) > Produit A (10s) > Produit B (10s) > Verdict (5s) > CTA (3s)

### Serie 5 : "Mythes skincare" (1x/semaine)
> Debunker les fausses croyances
- Format : 20-30 secondes
- Hook : "NON, [mythe populaire]"
- Structure : Mythe (3s) > Pourquoi c'est faux (15s) > La verite (5s) > CTA (3s)

### Serie 6 : "Le saviez-vous ?" (2x/semaine)
> Facts rapides sur les ingredients
- Format : 15-20 secondes
- Hook : "La vitamine C dans ton serum..."
- Structure : Fait surprenant (3s) > Explication (10s) > CTA (3s)

### Serie 7 : "Clean Beauty Score" (1x/semaine)
> Presenter le score clean beauty unique a ECLAT
- Format : 25-30 secondes
- Hook : "Ce produit a un score de [X]/100"
- Lie directement au site maison-eclat.shop

### Serie 8 : "Decode ton INCI" (1x/semaine)
> Apprendre a lire les etiquettes
- Format : 30-40 secondes
- Hook : "Tu sais lire ca ?" [montre une etiquette]
- Educatif, positionne ECLAT comme expert

### Serie 9 : "Avant/Apres" (1x/semaine)
> Resultats avec les produits ECLAT (photos clients avec permission)
- Format : 15-25 secondes
- Hook : "30 jours avec le [produit ECLAT]"

### Serie 10 : "Trending Sound + Beaute" (selon tendances)
> Surfer sur les sons viraux avec un angle beaute
- Format : variable
- Reagir vite aux tendances (dans les 24h)
