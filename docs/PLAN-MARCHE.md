### Stratégie prix "Newcomer Agressive"

**Règle d'or** : En tant que nouveaux, on ne peut PAS se permettre d'être chers. Notre arme = prix BAS + qualité HAUTE + UX PREMIUM.

**Formule de prix** :
```
Prix CJ × 2.5 à 4.0 = Prix de vente
```
- Produits d'appel (petits accessoires) : markup ×3-4 → prix 3.90-9.90€
- Produits coeur (sérums, crèmes) : markup ×3-3.5 → prix 9.90-19.90€
- Produits premium (appareils, kits) : markup ×2.5-3 → prix 19.90-49.90€
- Coffrets : markup ×2.5 → prix attractif vs achat séparé (-15 à -25%)

## Analyse Concurrentielle Détaillée

### Forces/Faiblesses des concurrents EU

#### Sephora / Douglas / Flaconi (Premium)
- **Forces** : Notoriété massive, confiance, marques connues, magasins physiques
- **Faiblesses** : Sites TRÈS lents (5-8s), prix élevés, fidélité opaque, zéro transparence ingrédients
- **Comment on gagne** : Prix 40-60% moins cher, site ultra-rapide (<2s), Clean Beauty Score, gamification
- **Présence** : FR (Sephora), DE (Douglas/Flaconi), ES (Sephora), IT (Sephora), UK (Boots/Sephora)

#### Amazon EU (toutes places de marché)
- **Forces** : Prix bas, livraison rapide (Prime), choix immense, confiance
- **Faiblesses** : Zéro expertise beauté, contrefaçons, pas de conseil, pas de communauté
- **Comment on gagne** : Expertise beauté, diagnostic IA, communauté, gamification, prix similaires

#### Notino / LookFantastic (Mid-range online)
- **Forces** : Pure players EU, prix compétitifs, large sélection
- **Faiblesses** : Pas de marque propre, pas d'innovation UX, pas de gamification
- **Comment on gagne** : Marque propre ÉCLAT, UX unique, Clean Beauty Score, fidélité innovante

#### Yves Rocher / Rituals / Kiko (Mid-range chaînes)
- **Forces** : Marques reconnues, naturel, prix accessibles, magasins
- **Faiblesses** : Image parfois vieillotte, pas de tendance TikTok, gamme limitée outils
- **Comment on gagne** : Plus moderne, plus tech, gamification, outils beauté que ces marques n'ont pas

#### Temu / AliExpress / Shein (Ultra-budget)
- **Forces** : Prix extrêmement bas, catalogue immense
- **Faiblesses** : Qualité douteuse, pas de SAV, délais longs, pas de confiance, scandales qualité
- **Comment on gagne** : Qualité vérifiée, SAV chatbot IA 24/7, site pro EU, conformité RGPD + CE

### Règles pour chaque produit

1. **Note CJ minimum 4.5/5** — Pas de produit mal noté
2. **Photos pro** — Uniquement produits avec belles photos fournisseur
3. **Certifications** — CE obligatoire pour appareils, INCI complet pour cosmétiques
4. **Pas de contrefaçon** — Jamais de copie de marque connue
5. **Ingrédients safe** — Pas de parabènes controversés, pas de formaldéhyde

7. **Délai raisonnable** — Max 15 jours de livraison (entrepôt EU préféré)
