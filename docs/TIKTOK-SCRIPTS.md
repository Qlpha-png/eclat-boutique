## Notes de production

### Conseils generaux :
- **Eclairage** : lumiere naturelle ou ring light, fond neutre ou salle de bain
- **Format** : 9:16 vertical, sous-titres obligatoires (80% des gens regardent sans son)
- **Rythme** : coupes rapides toutes les 3-5 secondes, zero temps mort
- **Authenticite** : parler face camera, pas de script lu, ton conversationnel
- **Musique** : tendances TikTok du moment, volume bas sous la voix

### Ordre de publication recommande :
1. Script 4 (provocateur, fort engagement)
2. Script 1 (introduit le Clean Beauty Score)
3. Script 2 (educatif, valeur ajoutee)
4. Script 8 (challenge interactif)
5. Script 5 (autorite)
6. Script 7 (educatif)
7. Script 3 (demo routine)
8. Script 6 (diagnostic IA)
9. Script 10 (routine builder)
10. Script 9 (fidelite, pour audience deja engagee)

### Metriques a suivre :
- Taux de visionnage complet (> 60% = bon hook)
- Commentaires (engagement)
- Clics lien bio (conversion)
- Partages (viralite)
