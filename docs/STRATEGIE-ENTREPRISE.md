## Positionnement Prix — Intelligent & Réaliste

### Philosophie de prix

```
On ne peut PAS vendre un sérum à 25€ sans notoriété.
On PEUT vendre un sérum identique à 9.90€ et gagner quand même 80% de marge.
```

### Matrice de positionnement

| Notre prix | Perception client | Type de produit |
|-----------|-------------------|-----------------|
| **2.90-4.90€** | "C'est donné !" — achat impulsif | Accessoires, baumes, limes |
| **5.90-9.90€** | "Bon rapport qualité/prix" — achat réfléchi | Patchs, masques, petits soins |
| **9.90-14.90€** | "Qualité à prix accessible" — achat confiant | Sérums, crèmes, outils |
| **14.90-24.90€** | "Deal incroyable" — achat excitant | Appareils, kits, coffrets |
| **24.90-49.90€** | "Professionnel abordable" — achat investi | Appareils pro, grands coffrets |

### Règle des seuils psychologiques

- **Jamais de prix rond** : 9.90€ pas 10€, 14.90€ pas 15€
- **Seuil d'achat** : Livraison gratuite dès 29€ (FR) / 35€ (EU)
- **Panier moyen cible** : 30-40€ (2-3 produits + 1 accessoire impulsif)

## Stratégie de Conquête en 5 Phases

### Phase A — "Le Lancement" (Mois 1-2)
**Objectif** : 100 premiers produits en ligne, 50 premières ventes.

Actions :

**KPIs** :
- 50 visiteurs/jour
- 1-2 ventes/jour
- 5 premiers avis positifs
- 3 vidéos TikTok à 1000+ vues

### Phase B — "La Crédibilité" (Mois 3-4)

**KPIs** :
- 200 visiteurs/jour
- 5 ventes/jour
- 20+ avis positifs
- 1 vidéo TikTok virale (10K+ vues)
- Taux de retour < 3%

### Phase C — "L'Expansion EU" (Mois 5-6)
**Objectif** : Ouvrir les marchés DE, ES, UK. 500 produits.

**KPIs** :

### Phase D — "La Croissance" (Mois 7-9)

**KPIs** :

- 20-30 ventes/jour
